## ğŸ—ƒï¸ Database Architecture

### **Database Type:** SQLite
**Database Name:** `CoffeeShop.db`  
**Database Version:** 1

### Database Tables:

#### 1. **Users Table**
Stores registered user information.
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    full_name TEXT NOT NULL,
    email TEXT UNIQUE NOT NULL,
    password TEXT NOT NULL
)
```

**Columns:**
- `id` - Unique user identifier
- `full_name` - User's full name
- `email` - User's email (unique, used for login)
- `password` - User's password (plain text for educational purposes)

#### 2. **Products Table**
Stores coffee and beverage product information.
```sql
CREATE TABLE products (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    description TEXT,
    price REAL NOT NULL,
    image_url TEXT
)
```

**Columns:**
- `id` - Unique product identifier
- `name` - Product name (e.g., "Espresso", "Cappuccino")
- `description` - Brief product description
- `price` - Product price in dollars
- `image_url` - Image resource name

**Pre-loaded Sample Data:**
- Espresso - $2.99
- Cappuccino - $4.49
- Latte - $4.99
- Americano - $3.49
- Mocha - $5.49
- Macchiato - $3.99
- Cold Brew - $4.99
- Flat White - $4.79

#### 3. **Cart Table**
Stores current shopping cart items for each user.
```sql
CREATE TABLE cart (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    product_id INTEGER NOT NULL,
    quantity INTEGER NOT NULL,
    FOREIGN KEY(user_id) REFERENCES users(id),
    FOREIGN KEY(product_id) REFERENCES products(id)
)
```

**Columns:**
- `id` - Unique cart item identifier
- `user_id` - Foreign key to users table
- `product_id` - Foreign key to products table
- `quantity` - Number of items

#### 4. **Orders Table**
Stores completed order transactions for revenue tracking.
```sql
CREATE TABLE orders (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    total_amount REAL NOT NULL,
    order_date TEXT NOT NULL,
    FOREIGN KEY(user_id) REFERENCES users(id)
)
```

**Columns:**
- `id` - Unique order identifier
- `user_id` - Foreign key to users table
- `total_amount` - Total order amount
- `order_date` - Timestamp of order (format: "yyyy-MM-dd HH:mm:ss")

## ğŸ”§ Supporting Components

### **SessionManager** (`SessionManager.java`)
Singleton class for managing user session.

**Properties:**
- `int userId` - Current logged-in user ID
- `String userName` - Current user's name

**Methods:**
- `getInstance()` - Get singleton instance
- `getUserId()` - Get current user ID
- `setUserId(int)` - Set user ID
- `getUserName()` - Get user name
- `setUserName(String)` - Set user name
- `isLoggedIn()` - Check if user is logged in
- `logout()` - Clear session data

### **DatabaseHelper** (`DatabaseHelper.java`)
Extends SQLiteOpenHelper for database operations.

**Key Methods:**

**User Operations:**
- `registerUser(name, email, password)` - Create new user
- `loginUser(email, password)` - Authenticate user
- `isEmailExists(email)` - Check email availability

**Product Operations:**
- `getAllProducts()` - Get all products
- `getProductById(id)` - Get specific product
- `addProduct(Product)` - Create new product
- `updateProduct(Product)` - Update existing product
- `deleteProduct(id)` - Delete product
- `searchProducts(query)` - Search by name

**Cart Operations:**
- `addToCart(userId, productId, quantity)` - Add/update cart item
- `getCartItems(userId)` - Get user's cart
- `updateCartItemQuantity(cartId, quantity)` - Update quantity
- `removeFromCart(cartId)` - Remove cart item
- `clearCart(userId)` - Empty user's cart

**Order Operations:**
- `createOrder(userId, totalAmount)` - Create order record
- `getTotalRevenue()` - Get all-time revenue
- `getRevenueByDate(date)` - Get daily revenue
- `getRevenueByMonth(year, month)` - Get monthly revenue
- `getRevenueByYear(year)` - Get yearly revenue

### **ProductAdapter** (`ProductAdapter.java`)
RecyclerView adapter for product list.

**Interface:** `OnProductClickListener`
- `onProductClick(Product)` - View details
- `onEditClick(Product)` - Edit product
- `onDeleteClick(Product)` - Delete product
- `onAddToCartClick(Product)` - Add to cart

**Features:**
- Binds product data to views
- Handles image resource mapping
- Manages click listeners for all buttons

### **CartAdapter** (`CartAdapter.java`)
RecyclerView adapter for cart items.

**Interface:** `OnCartItemListener`
- `onQuantityChanged(CartItem, newQuantity)` - Update quantity
- `onRemoveClick(CartItem)` - Remove item

**Features:**
- Displays cart items with quantities
- Calculates subtotals
- Handles quantity increase/decrease
- Manages item removal
